About

Mukti is a powerful first-person story exploration game set in an Indian museum, tackling the critical issue of human trafficking. Players journey through winding corridors, uncovering hidden stories and harsh realities faced by victims. With immersive gameplay and emotional storytelling, Mukti sheds light on this pressing global crisis.

Project Info

  • Role: Game programmer
  • Team Size: 15
  • Time frame: 3 months
  • Engine: Unreal (C++)

Project Overview

This project was selected for the India Hero Project, an initiative by Sony that offers mentorship on building industry-standard production pipelines, along with access to internal resources from Sony Interactive. I was fortunate to be a part of this program, where I had the opportunity to work directly with the Unreal Engine source code provided by Epic Games.

During the project, I implemented adaptive triggers and haptic feedback by converting Sony’s internal APIs into Unreal Engine. I also contributed to the development of puzzle systems and player interaction mechanics using new Input system with different controls.

Due to the nature of the program and confidentiality agreements, I’m unable to disclose further details at this time.
