About

A narrative based story game where you not only read through the story but is also interactive where you can make a choice and decide the path of the story. You can lead the story as your choices can decide the end of the story.

Project Info

  • Role: Game Programmer
  • Team Size: 9
  • Time frame: 1 years
  • Engine: Unity (C#)

Overview

This was my first professional project, where I joined Underdogs Studio as a Junior Game Programmer. I had the opportunity to work alongside incredibly talented developers and receive valuable mentorship from senior team members.

During my initial month,I explored new features and techniques while learning the whole development pipeline. This allowed me to contribute meaningfully early on including proposing a new set of rules to streamline and improve the story flow system.

These are some of the key features I worked on:

  • Addressables & AWS integration: Used Unity Addressables to upload story bundles to AWS, letting the app download episode assets on demand so users only download what they play.
  • Extended Fungus Visual Scripting: Built custom editor extensions for Fungus, adding new node types and utilities to the flowchart interface so non-programmers can craft complex dialogue branches and events without touching code.
  • Automated Doc‐to‐Interactive Story: Contributed in automatic the process of converting story docs files to working stories in unity where docs were color coded by designers which then is converted in csv file with commands and signs which then used in unity to generate story flow
  • UI programming: Implemented in-game interfaces for navigating episodes, displaying dialogue, and handling player choices.
